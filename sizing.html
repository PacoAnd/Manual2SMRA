Entendiendo Box-Sizing en CSS

Esta es una guía para comprender cómo funciona la propiedad CSS box-sizing en los elementos HTML.

¿Qué es box-sizing?

La propiedad box-sizing en CSS determina cómo se calculan las dimensiones de los elementos HTML (su ancho y alto). Por defecto, el tamaño de los elementos incluye solo el área de contenido, pero con box-sizing podemos cambiar esta regla para que también se incluyan el borde y el relleno en las dimensiones totales del elemento.

Valores de box-sizing

Existen dos valores principales que puede tomar la propiedad box-sizing:

  • content-box: Este es el valor por defecto. El tamaño del elemento solo incluye el contenido, y el borde y el relleno se agregan fuera de ese tamaño.
  • border-box: En este caso, el tamaño del elemento incluye el contenido, el borde y el relleno, lo que hace que el tamaño total sea más fácil de controlar.

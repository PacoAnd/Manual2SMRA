Manual sobre los Distintos Layouts en CSS

Este manual explica cómo crear diferentes tipos de diseños o "layouts" utilizando CSS. Incluye ejemplos con block, flexbox, grid y cómo evitar el desplazamiento horizontal no deseado.

¿Qué es un Layout en CSS?

En CSS, un layout es el modelo que define la disposición de los elementos en la página. Los tres layouts más comunes son:

  • Layout de Bloques (block): Los elementos se apilan verticalmente ocupando el ancho completo de su contenedor.
  • Layout Flexbox: Un modelo de caja flexible que facilita la alineación y distribución de los elementos dentro de un contenedor.
  • Layout Grid: Un sistema de diseño basado en una cuadrícula de filas y columnas.

Evitar el Desplazamiento Horizontal

Es importante asegurarse de que el contenido se ajuste al ancho de la pantalla sin causar desplazamiento horizontal. Para evitarlo:

  • Usa unidades relativas como vw o % para los anchos.
  • Utiliza box-sizing: border-box; para que los márgenes y rellenos no aumenten el tamaño total de los elementos.
  • Evita los anchos fijos que superen el 100% del ancho de la pantalla.

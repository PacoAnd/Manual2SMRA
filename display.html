Manual sobre la Propiedad CSS display

Este manual explica cómo funciona la propiedad CSS display, los valores más comunes y su impacto en el diseño de los elementos HTML.

¿Qué es la propiedad display?

La propiedad display determina cómo se comporta un elemento dentro del flujo del documento y cómo se renderiza en el navegador.

Los valores más comunes son:

  • block: El elemento ocupa todo el ancho disponible y comienza en una nueva línea.
  • inline: El elemento ocupa solo el espacio necesario y se sitúa en la misma línea que otros elementos.
  • inline-block: Combina características de inline y block.
  • none: Oculta el elemento, eliminándolo del flujo del documento.
  • flex: Convierte el contenedor en un modelo flexible para alinear y distribuir elementos.
  • grid: Convierte el contenedor en un modelo de cuadrícula con filas y columnas.

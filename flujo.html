Manual sobre el Flujo de Elementos en el Navegador

Este manual explica cómo los elementos HTML interactúan dentro de una página web, dependiendo de su modelo de diseño, sus propiedades CSS y el flujo del documento.

Flujo Normal de los Elementos

El flujo de los elementos define cómo se colocan en la ventana del navegador de manera predeterminada. Existen dos tipos principales de flujo:

  • Elementos en bloque: Ocupan todo el ancho disponible y se apilan uno debajo del otro. Ejemplos: <div>, <p>, <h1>.
  • Elementos en línea: Ocupan solo el espacio necesario para su contenido y se colocan uno al lado del otro en la misma línea. Ejemplos: <span>, <a>, <strong>.

Propiedades que Afectan el Flujo

Algunas propiedades CSS pueden modificar el flujo normal de los elementos, alterando su comportamiento y disposición:

  • display: Define si un elemento es bloque, en línea, flex, grid, etc.
  • float: Permite alinear elementos a la izquierda o derecha, sacándolos del flujo normal.
  • clear: Evita que un elemento se posicione junto a elementos flotantes.
  • position: Cambia la posición de un elemento (estática, relativa, absoluta, fija o sticky).
  • flexbox: Organiza elementos en un diseño flexible de filas o columnas.
  • grid: Organiza los elementos en un diseño de cuadrícula con filas y columnas.
